# Changelog

All notable changes to UDP Trigger will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Home screen widget for quick access
- Auto-reconnect to last destination on startup
- Privacy policy in About dialog
- Comprehensive About screen with version info, build info, and open source licenses

### Changed
- Enhanced error handling and user-friendly error messages

## [1.0.0] - 2025-01-22

### Added
- Ultra-low latency UDP packet transmission
- Any-key trigger support (volume buttons, media keys, USB keyboard)
- IPv4, IPv6, hostname, and broadcast address support
- Configurable connection presets for quick switching
- Background service for continuous operation
- Connection health monitoring with visual indicators
- Latency visualization with history graph
- Burst mode for rapid packet transmission
- Scheduled triggers with configurable intervals
- Listen mode for receiving UDP packets
- Auto-reconnect on network changes
- Haptic and sound feedback options
- JSON export/import of settings and packet history
- Onboarding tutorial for first-time users
- Material 3 design with dynamic color support (Android 12+)
- Light and dark theme support

### Security
- No data collection or analytics
- All settings stored locally on device
- No third-party services or cloud dependencies
